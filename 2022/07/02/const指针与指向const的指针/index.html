const指针与指向const的指针

首先要明白两个原则：
1）指针符号“*”与标志符结合，而不是与数据类型结合。
2）const总是放在被修饰者的右边（即总是修饰左边）。

1, 指向const的指针

const int *u;：指向const的指针，它的意思是：这个指针指向的值（存储块）是不能改变的。如何更好理解它？

我们可以将const与int调换位置，等价写法为：int const *u;。根据上面两个原则可知：

1）const修饰的是int类型，即不变的是指针指向的值。
2）*u是一个普通的指针，指针的值可变。

2, const指针

int* const u;：const指针，它的意思是：这个指针的值不变。

1）const修饰它左边的符号“*”，即指针的值不变。
2）指针指向的值可以改变。
